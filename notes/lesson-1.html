Arrays

  • Arrays are are a type of object that are designed with the usage of square brackets.
  • Length is a helpful attribute.
  • Objects can have functions, (example: push function) which are invoked with parentheses.
  var fruits = ["kiwi", "strawberry", "plum"]
  var howLong = fruits.length
  var colors = ["blue", "green", "gold", "silver"]
  var addBronze = colors.push("bronze")
  var sliceIt = fruits.slice(2)

Arrays and forEach

  sliceIt.forEach(function(metal) {
  console.log('Metal colors: ' + metal)
  })
